# Pet Adoption Website

This is my first project
A responsive pet adoption webpage built using semantic HTML and CSS.  
The project is designed to showcase pets available for adoption with a clean card-based layout and user-friendly interface.

## Features
- Semantic HTML structure
- Card-based layout for pet profiles
- Hover effects and smooth transitions
- Responsive design with media queries
- Feedback and contact section

## Tech Stack
- HTML5
- CSS3
- Responsive Design

## How to Run
1. Clone the repository
2. Open `pet_adoption.html` in a web browser

## Learning Outcomes
- Improved understanding of semantic HTML elements
- Practical experience with CSS layouts and transitions
- Basic responsive design using media queries
